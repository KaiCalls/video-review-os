# KaiCalls Mascot Overlay Demo

This example shows the branded frame-overlay workflow.

Inputs:

- `../../assets/kai-mascot-hero.png`
- `../../assets/sample-frame.svg`
- `project/scenes.json`
- `project/drafts/copy.json`

Generated outputs:

- `project/visuals/thumbnails/clip-001.svg`
- `project/visuals/scene-cards/clip-001.svg`
- `project/visuals.json`

The SVG outputs embed the frame and mascot as data URIs, so they can be viewed directly in GitHub.
