# Example Assets

This folder contains visible demo assets for Video Review OS examples.

- `kai-mascot-hero.png` is the KaiCalls mascot example used to demonstrate frame overlays.
- `sample-frame.svg` is a synthetic sample frame used by the overlay demo.

The pipeline itself is not tied to these assets. Set `mascot_image_path` and `logo_image_path` in your config to use your own transparent PNG assets.
